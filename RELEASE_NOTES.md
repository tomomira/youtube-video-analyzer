# YouTube Video Analyzer - Release Notes

## Version 1.0.0 (2025-11-27)

### 🎉 初回リリース

YouTube Video Analyzer v1.0.0をリリースしました！このアプリケーションは、YouTube Data API v3を使用して動画を検索し、結果をExcelやGoogle Spreadsheetsにエクスポートできるデスクトップアプリケーションです。

---

## ✨ 主な機能

### 検索機能
- ✅ **キーワード検索**: 自由なキーワードで動画を検索
- ✅ **再生回数フィルター**: 最小・最大再生回数で絞り込み
- ✅ **動画タイプフィルター**: ショート動画（60秒以下）と通常動画を区別
- ✅ **ソートオプション**: 再生回数順、公開日順、評価順などで並び替え
- ✅ **最大取得件数指定**: 1〜500件まで設定可能

### 結果表示
- ✅ **テーブル表示**: 動画情報を見やすいテーブルで表示
- ✅ **列ソート**: 各列をクリックして昇順・降順でソート
- ✅ **動画情報**: タイトル、チャンネル名、再生回数、動画の長さ、公開日、種類（ショート/通常）
- ✅ **URLアクセス**: 行をダブルクリックでブラウザで動画を開く

### エクスポート機能
- ✅ **Excelエクスポート**: .xlsx形式で保存、ヘッダー整形、ハイパーリンク付き
- ✅ **Google Sheetsエクスポート**: クラウド上のスプレッドシートに直接書き込み、URLを自動でクリップボードにコピー

### プリセット機能
- ✅ **検索条件の保存**: よく使う検索条件をプリセットとして保存
- ✅ **プリセット読み込み**: 保存したプリセットをワンクリックで適用
- ✅ **プリセット管理**: プリセットの削除、更新が可能
- ✅ **SQLiteデータベース**: プリセットはローカルDBに永続化

### 検索履歴機能
- ✅ **履歴の記録**: 検索条件と結果件数を自動で記録
- ✅ **履歴から再検索**: 過去の検索条件で再度検索を実行
- ✅ **履歴検索**: キーワードで履歴を検索
- ✅ **履歴管理**: 個別削除、全削除が可能

## 📦 インストール方法

### 方法1: スタンドアロン実行可能ファイル（推奨・初心者向け）

**注意**: 現在準備中です。将来的にリリースページから実行可能ファイルをダウンロード可能になります。

### 方法2: Pythonソースコードから実行（現在の方法）

1. **リポジトリをクローン**
   ```bash
   git clone https://github.com/your-repo/youtube-video-analyzer.git
   cd youtube-video-analyzer
   ```

2. **仮想環境を作成**
   ```bash
   python -m venv venv
   source venv/bin/activate  # macOS/Linux
   venv\Scripts\activate     # Windows
   ```

3. **依存パッケージをインストール**
   ```bash
   pip install -r requirements.txt
   ```

4. **環境変数を設定**
   ```bash
   cp .env.example .env
   # .envファイルを編集してYouTube APIキーを設定
   ```

5. **アプリケーションを起動**
   ```bash
   python src/main.py
   ```

詳細は [インストールガイド](./docs/installation-guide.md) を参照してください。

## 🐛 既知の問題

### 制限事項

1. **APIクォータ制限**:
   - YouTube Data API v3は1日10,000ユニットまで
   - 検索しすぎるとクォータ超過エラーが発生
   - 翌日になればリセットされます

2. **ショート動画の識別**:
   - 60秒以下を「ショート動画」と判定
   - YouTube公式のショート動画フラグではないため、正確性に限界があります

3. **Google Sheetsエクスポート**:
   - サービスアカウントの設定が必要
   - 初回セットアップがやや複雑

### トラブルシューティング

**問題**: 検索結果が0件
- **解決策**:
  - キーワードを変更してみる
  - 検索条件（再生回数範囲）を緩くする
  - APIクォータが残っているか確認

**問題**: 「YouTube API キーが設定されていません」エラー
- **解決策**:
  - `.env`ファイルにAPIキーが正しく設定されているか確認
  - `.env`ファイルがプロジェクトのルートディレクトリにあるか確認

詳細は [インストールガイド - トラブルシューティング](./docs/installation-guide.md#トラブルシューティング) を参照してください。

## 🔐 セキュリティ

### 推奨事項

1. **APIキーの管理**:
   - `.env`ファイルは絶対に公開しない
   - GitHubなどにコミットしない（`.gitignore`で除外済み）
   - APIキーは定期的にローテーション

2. **Google Sheets認証情報**:
   - サービスアカウントのJSONキーファイルも公開しない
   - 必要最小限の権限のみ付与

3. **データベース**:
   - `youtube_analyzer.db`にはAPIキーは保存されません
   - プリセットと検索履歴のみ保存
